# Script de démo (5 minutes max)

## 0:00–0:30 — Ouverture
- Message: "On vous montre une restitution prête sponsor, de la donnée brute à l'action." 
- Montrer rapidement les livrables `executive_summary.md` + `one_pager_comex.md`.

## 0:30–1:30 — Accueil Streamlit
- Ouvrir page **Accueil**.
- Verbaliser:
  1) adoption déjà engagée,
  2) maturité hétérogène,
  3) besoin de gouvernance + formation.
- Mettre en avant les KPI de tête (taille échantillon, usage pro, usage perso).

## 1:30–2:30 — Vue d’ensemble
- Ouvrir page **Vue d’ensemble**.
- Montrer les 4 graphiques retenus (usage, rôles, 2 segmentations).
- Message clé: "Peu de bruit visuel, seulement les visuels utiles à la décision." 

## 2:30–3:30 — Segmentations
- Ouvrir page **Segmentations**.
- Insister sur:
  - expertise ↔ usage pro (signal robuste),
  - usage perso ↔ usage pro (signal robuste),
  - rôle ↔ usage (non concluant statistiquement).
- Mentionner explicitement: "association ≠ causalité".

## 3:30–4:30 — Verbatims + recommandations
- Ouvrir **Verbatims** puis **Recommandations**.
- Montrer: irritants, attentes de gouvernance, idées business, besoins d'accompagnement.
- Enchaîner avec les 5 actions priorisées et les propriétaires proposés.

## 4:30–5:00 — Clôture
- Conclusion en 3 phrases:
  1. "L'adoption est déjà là; l'enjeu est d'industrialiser sans perdre le contrôle."
  2. "Le package combine preuves chiffrées, lecture managériale et plan d'action." 
  3. "La décision attendue: valider le cadre, lancer la formation, et prioriser 3 cas d'usage 2026."
