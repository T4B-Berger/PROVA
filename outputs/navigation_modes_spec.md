# Navigation modes spec

## Toggle global
- `Description seule : Oui / Non`

## Mode Description seule = Oui
Visible:
- Accueil
- Cockpit COMEX factuel
- Maturité
- Segmentations
- Réponses individuelles
- Verbatims intelligents
- Artefacts descriptifs

Règles:
- faits observés uniquement
- regroupements descriptifs
- tags/fréquences explicables
- aucune recommandation, aucun owner, aucun sponsor proposé

## Mode Description seule = Non
Visible:
- toutes les vues descriptives
- + Use case lab
- + Gouvernance
- + Enablement
- + Roadmap / Action tracker
- + Recommandations / priorisation

Règles:
- les contenus prescriptifs sont marqués explicitement: recommandation / proposition
- séparation claire entre constat et projection
